⚡ CAMBIOS RÁPIDOS (SIN PROGRAMAR)

   Cambiar velocidad de spawn:
   Edita config.js → documents.spawnInterval → (en ms)

   Cambiar tema visual:
   Abre consola (F12) → applyTheme('dark')

   Cambiar dificultad:
   Abre consola (F12) → applyDifficulty('hard')

   Cambiar costos de mejoras:
   Edita config.js → upgrades → baseCost

   Cambiar colores:
   Edita config.js → THEMES → selecciona tema

🔌 COMANDOS ÚTILES EN CONSOLA (F12)

   game.money += 1000              → Agregar 1000 dinero
   game.activatePowerup('shredder') → Activar trituradora
   applyTheme('neon')              → Cambiar tema a neon
   applyDifficulty('hard')         → Cambiar a dificultad difícil
   game.spawnDocument()            → Generar un documento
   game.buyUpgrade('assistant')    → Comprar mejora
   game.saveGame()                 → Guardar manualmente
   game.resetGame()                → Reiniciar juego
   window.game                     → Ver objeto del juego

❓ PROBLEMAS COMUNES

   P: El juego no carga
   R: Abre F12, busca errores rojos, recarga (Ctrl+F5)

   P: El juego va lento
   R: Cierra otras pestañas, limpia caché

   P: No se guardan los datos
   R: No uses modo incógnito, usa navegador moderno

   P: Las mejoras son muy caras
   R: Cambia dificultad → applyDifficulty('peaceful')

   P: Quiero más documentos
   R: En config.js → documents.spawnInterval = 1000 (menos = más)

   Más ayuda: Ver CHANGELOG.md → FAQ
